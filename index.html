Introduction

Ant is a PHP framework for rapid web application development, designed to be lightweight and to have a consistently intuitive MVC-based structure that is as clear in its directory structure as it is in the code.

Features Backlog

  • Ability to scale from tiny projects to hundreds of files.
  • Keep PHP files small by relying heavily on lazy-loading.
  • Flexible routing in XML files
  • Separation of PHP and HTML using optimized string replacement (avoiding PHP's regex functions)
  • Built in handling of JavaScript and Less.css compiling, optimization.
  • Built in basic database structure that every common application needs
  • Basic handling of user functionality, like login and authentication through Facebook/Google API
  • Separation of database queries, to allow for reusability; ability to extend queries easily.
  • Accessing of controllers directly via whitelisted http requests.
  • Internalization of languages.
  • Client detection (mobile, tablet, browser) and display of different/partially different views.
  • "Channelling" option, to change output based on the same route.
